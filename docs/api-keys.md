# API Keys and Secrets

Add secrets in these folders:

- `backend/.env`
  - `DATABASE_URL` (PostgreSQL connection string)
  - `JWT_SECRET`
  - `SUPABASE_URL` (Phase 3)
  - `SUPABASE_SERVICE_ROLE_KEY` (Phase 3)

- `mobile/.env`
  - `API_BASE_URL`
  - `SUPABASE_URL` (Phase 3)
  - `SUPABASE_ANON_KEY` (Phase 3)
